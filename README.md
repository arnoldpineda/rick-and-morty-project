# Rick and Morty Project

## Backend
- Node v18.17.1
- Npm v6.14.18
- Typescript
- Express
- MySQL
- Redis
- GraphQL

## Frontend
- React 18
- React Router DOM
- TailwindCSS
- GraphQL


## Install
```sh
npm install
```

## Run in dev mode

```sh
npm run dev
```

## Run in prod mode

```sh
npm run start
```
